## 随机音乐播放
API整合菜鸟整理出来的项目

### 开发
- Golang
- Gin框架
- HTML5
- CSS3
- JavaScript

### 项目介绍
API调用的是网易云音乐的歌曲排行榜， 支持四种榜单的随机音乐播放。程序支持自动播放和播放完成后切换歌曲，想怎么改就怎么改吧。我也是学习开发的，

### 构建项目

- 安装库， 需要的库在go.mod里面，自行下载缺失库

- 构建项目

  ```go
  go build
  ```

- 编译各种平台版本请自行百度：Go交叉编译

- 运行：请将编译文件以及views目录一同上传服务器然后再运行，否则报错

- 授予权限并运行

  ```shell
  chmod g=rwx 编译文件
  ./编译文件
  ```

  

- 长久运行可以搭配Supervisord使用
